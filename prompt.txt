## OUTPUT VISUALIZATIONS

Build these 4 visualization panels after AI response:

### Panel 1: Career Match Cards
- Top 3 recommended careers as visual cards
- Each card shows: Domain icon, Match % circular gauge, Entry route, Salary range
- Color coded: Green (>80% match), Yellow (60-80%), Orange (<60%)
- Expandable to show full details

### Panel 2: Interactive Decision Tree
- SVG-based visual decision tree for the top recommended path
- Nodes: Current position → Entrance Exam → College Type → Specialization → Career
- Clickable nodes that expand to show details
- Color: Blue (completed/current) → Green (recommended) → Gray (alternative)

### Panel 3: AP/TS Domain Supply-Demand Chart
- Horizontal bar chart (use Chart.js from CDN)
- For each domain: show Demand score vs Supply score as grouped bars
- Color the "opportunity gap" — wider gap = more opportunity
- Add trend arrows (↑↓→)
- Label with data source

### Panel 4: Budget vs Path Feasibility Matrix
- Table showing each recommended path vs budget buckets
- Traffic light: Green = feasible, Yellow = stretch with scholarship, Red = out of range
- Show scholarship options that bridge the gap
